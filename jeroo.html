Diamonds

Description

Nets

Description

I used Jeroo to clear the objects in my way by picking up flowers and placing them where the nets were. The end goal was to finish where the final net was.

Concepts Learned

  • I learned to use the toss command and that when you toss a flower on a net, the net disappears

Great Escape

Description

I had to create a single method that allowed my Jeroo to navigate through the complex maze and make the Jeroo stop on the flower and pick it up. Each Jeroo used the same method to safely navigate through the maze without touching the nets.

Concepts Learned

  • I learned how to use methods
  • I learned how to apply a single method for several Jeroos

LED

Description

I created a method that would allow the Jeroo to plant flowers to form any number from 0 to 9 in a 5 column by 7 row format. Each method had to start from the top left and finish at the top row along the right edge. Then I wrote the code to plant the last 3 numbers of my phone number on the screen.

Concepts Learned

  • I learned how to use several methods in one main method
  • I learned how to reuse parts of a method to create new ones

Spiral Run

Description

I created a method that would allow my Jeroo to spiral from the start to the middle of the island where the flower is located. The Jeroo finished facing the flower.

Concepts Learned

  • I learned how to use while loops
